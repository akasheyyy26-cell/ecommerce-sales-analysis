# 🛒 E-Commerce Sales Analysis

Analyzing 9,993 orders to uncover business insights using Python & Pandas.

---

## 📌 Business Problem
An e-commerce company wants to understand:
- Which categories and regions drive the most revenue?
- How do sales trend over time?
- Where are the highest profit opportunities?

---

## 📊 Key Insights
| Metric | Value |
|--------|-------|
| 💰 Total Revenue | $11.08 Million |
| 📈 Total Profit | $1.04 Million |
| 🛒 Total Orders | 9,993 |
| 📦 Avg Order Value | $1,109 |
| 💹 Profit Margin | 9.4% |

- 🏆 **Technology** is the top revenue category at $3.93M
- 🌍 **West region** leads all regions in sales
- 👥 **Consumer segment** drives 50% of total revenue
